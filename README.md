# 🚨 Error 404: Good Design Not Found

Welcome to UI/UX Purgatory. This is a curated exhibition of the world's most agonizing design disasters, translation failures, and cursed interfaces. Built purely out of spite for bad design.

### 🌐 Live Website
**[Klicke hier, um die Live-Seite zu sehen!](https://spimur.github.io/ERROR_404_Blog/)**

### 🛠️ Technologies Used
This project was built from scratch without any heavy frameworks to fulfill the requirements of a native web engineering project at TBZ:
- **HTML5 & CSS3** (Semantic tags, Custom Properties, Flexbox, CSS Grid)
- **Vanilla JavaScript** (DOM Manipulation, Web Storage API, async/fetch)
- **APIs:** - Formspree (for backendless email routing)
  - ImgBB API (for anonymous client-side image uploads via AJAX)
- **Design Language:** Neo-Brutalism (Hard shadows, monospace typography, high contrast)
